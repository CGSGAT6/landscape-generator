# landscape-generator
Application for university programming project

## Установка

Склонировать проект там где будет директория с проектом:

`git clone <URL>`

Внутри `landscape-generator`:

```
python -m venv .venv

.venv\Scripts\activate

pip install -e ".[dev]"
```

Можно работать

Для запуска используйте `demo.py`
